# FTP Downloader

یک ابزار دسکتاپ ساده و سبک برای ویندوز که دانلود فایل از سرور FTP را با صف دانلود، تاخیر بین فایل‌ها، و مرور مسیرهای FTP در اختیار کاربر قرار می‌دهد.

## معرفی پروژه

این پروژه با `C#` و `Windows Forms` ساخته شده و هدف آن ارائه‌ی یک برنامه‌ی کاربردی برای دانلود چند فایل از سرور FTP به‌صورت مرحله‌ای و قابل کنترل است. این پروژه می‌تواند هم به‌عنوان یک ابزار واقعی برای استفاده‌ی روزمره مفید باشد و هم به‌عنوان نمونه‌کار برای نمایش مهارت در توسعه‌ی اپلیکیشن دسکتاپ با .NET.

## امکانات

- اتصال به سرور FTP با نام کاربری و رمز عبور
- پشتیبانی از فعال یا غیرفعال کردن SSL
- افزودن چند فایل به صف دانلود
- تنظیم تاخیر بین دانلود فایل‌ها
- مرور دایرکتوری FTP و انتخاب فایل‌ها از داخل برنامه
- نمایش وضعیت هر فایل در حین دانلود
- نمایش پیشرفت کلی دانلود
- امکان توقف دانلود در هر زمان

## فناوری‌های استفاده‌شده

- `C#`
- `.NET 8`
- `Windows Forms`
- `FtpWebRequest`

## ساختار پروژه

- `FTP/Program.cs`: نقطه‌ی ورود برنامه
- `FTP/MainForm.cs`: رابط اصلی و مدیریت جریان دانلود
- `FTP/FtpDownloadService.cs`: منطق ارتباط با FTP و دانلود فایل‌ها
- `FTP/FtpBrowseDialog.cs`: پنجره‌ی مرور فایل‌های FTP
- `FTP/DownloadItem.cs`: مدل هر فایل در صف دانلود
